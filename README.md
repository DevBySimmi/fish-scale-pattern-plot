# 🐟 Fish Scale Pattern Plot

A colorful and geometric fish-scale-inspired pattern created using Python, NumPy, and Matplotlib.

This project uses overlapping circular arcs to create a repeating scale-like visual pattern with vibrant colors and transparency.
